Accept cryptocurrency payments directly to your wallet. Non-custodial, no API keys required.

== Description ==

SafePay Crypto Gateway allows your WooCommerce store to accept cryptocurrency payments with zero complexity:

* **Non-Custodial**: Payments go directly to your wallet. We never hold your funds.
* **100+ Cryptocurrencies**: Accept BTC, ETH, LTC, XRP, DOGE, SOL, and many more.
* **Stablecoin Settlement**: Receive payments in USDC or USDT - no volatility risk.
* **Zero API Keys**: No registration or API keys required from external services.
* **Best Rates**: We rate-shop across multiple exchanges for the best conversion rates.
* **Beautiful Checkout**: Modern, responsive checkout page with QR codes.

= How It Works =

1. Customer selects "Pay with Crypto" at checkout
2. Customer chooses their preferred cryptocurrency
3. Customer sends payment to the displayed address
4. Payment is automatically converted and sent to your wallet
5. Order is marked as paid

= Requirements =

* WordPress 5.8 or higher
* WooCommerce 6.0 or higher
* PHP 7.4 or higher
* HTTPS enabled on your site

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wc-crypto-gateway/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce → Settings → Payments → SafePay Crypto
4. Enter your wallet address and select your preferred stablecoin
5. Click "Register Store" to get your API credentials automatically
6. Save settings and start accepting crypto!

== Frequently Asked Questions ==

= What cryptocurrencies can customers pay with? =

Over 100 cryptocurrencies including Bitcoin, Ethereum, Litecoin, XRP, Dogecoin, Solana, and many more.

= Do I need to create an account or get API keys? =

No! SafePay uses public APIs that require no registration. Just install the plugin and enter your wallet address.

= Is my money safe? =

Yes! SafePay is non-custodial, meaning we never hold your funds. Payments are converted and sent directly to your wallet.

= Which networks are supported? =

USDC and USDT on Ethereum, Tron, BNB Smart Chain, Polygon, Solana, Arbitrum, Avalanche, and Optimism.
